## 项目分析（头脑风暴）

### 1. 现有背词软件的不足
>* 背词清单没有科学规划，学生背词效率差。
>* 功能单一，背词十分枯燥。
>* 界面繁杂，容易让使用者开小差。
>* 有些好的功能收费，让部分用户无法使用。

### 2. 改进方式

- 科学规划背词清单，提高背词效率。

- 除了背词添加一些其他例如文章推荐功能，让用户能在具体情境中背单词。

- 制作更加简洁的界面。

- 所有功能全部免费。

### 3. 单词来源

* 网上的一些资源，根据我们的筛选，选择更加重要的单词

### 4. 需求


* 登陆

* 背词

* 复习

* 文章推荐

* 个人信息管理

* 查询单词

* 进步数据可视化

* 好友排行

* 等等
